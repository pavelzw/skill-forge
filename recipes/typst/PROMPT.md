## Structure

- `SKILL.md` - Main skill file with core Typst syntax and overview
- `references/SYNTAX.md` - Complete syntax reference
- `references/SCRIPTING.md` - Scripting and programming guide
- `references/STYLING.md` - Set rules, show rules, and styling
